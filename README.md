# view-ignored

Get a list of ignored files by targets: git, npm, yaml, docker, ... .

## Features

- View a list of ignored and included files.
- Sorting a list.

### Targets

- Git (.gitignore, git config get core.excludesFile)

Broken:

- NPM
- Yarn
- VSC Extension

Planned:

- pnpm
- ESLint
